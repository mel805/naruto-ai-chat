## ğŸ› Troubleshooting

### "Invalid API Key"
- Make sure key starts with `gsk_`
- Check for spaces before/after
- Verify key is active: https://console.groq.com/keys

### "Rate limit exceeded"
- You've exceeded 14,400 requests/day
- Wait 24 hours
- Or create another Groq account with different email

### App crashes
- Make sure Android 8.0+ (API 26)
- Check logs: `adb logcat | grep Naruto`
- [Report issue](https://github.com/mel805/naruto-ai-chat/issues)

### Connection fails
- Check internet connection
- Verify API key in Settings
- Test on Groq Playground: https://console.groq.com/playground
